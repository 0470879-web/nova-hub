{{mask}}{{GitHub Codespaces}}

{{mask}}{{Setup Instructions}}
- {{mask}}{{Fork this repository to your GitHub account}}
- {{mask}}{{Head to the official}} Codespaces website
- Select New Codespaces and find your fork
- Ensure the branch is set to master
- Select Create Codespace
- Type npm run fresh-install and npm start
- Click "Make public" on the popup, then access via the ports tab
